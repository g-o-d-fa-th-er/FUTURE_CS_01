# Future Interns - Cyber Security Internship
## Task 1: Web Application Penetration Testing

**Target:** OWASP Juice Shop (Local Instance)
**Tools:** Kali Linux, Burp Suite, OWASP ZAP

### Description
This repository contains the findings from my vulnerability assessment of the OWASP Juice Shop application. I successfully identified and exploited three critical vulnerabilities:
1. SQL Injection (Admin Access)
2. Reflected XSS
3. Broken Access Control (Scoreboard Access)

### Files Included
- **Vulnerability Assessment.pdf**: Full detailed report with remediation steps.
- **Screenshots**: Proof of concept images for each vulnerability.
